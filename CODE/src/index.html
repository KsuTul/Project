История компании

Компания была создана в 2019 году для дипломной работы по результатам курса HTML+CSS. Профиль компании - строительство жилых домов из алюминия под ключ.

Почему выбирают нас

В даный момент строительство домов из алюминия представлено только компанией AlHome. Конкуренция идёт лишь с компаниями, строящими дома из традиционных материалов.
